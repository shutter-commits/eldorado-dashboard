A Eldorado usa autenticação via AWS Cognito — não há uma API Key fixa. Você se autentica com seu e-mail e senha da conta Eldorado.

⚠️ Pré-requisito: você precisa primeiro enviar um e-mail para api@eldorado.gg pedindo autorização de acesso à API. Após aprovação, use suas credenciais abaixo.
