# 📱 Como instalar o Eldorado Dashboard como App

## Arquivos necessários (todos na mesma pasta)
```
eldorado-painel.html
manifest.json
service-worker.js
icon-192.png
icon-512.png
```

---

## 🌐 Opção 1 — Hospedar no Vercel (recomendado)

1. Acesse https://vercel.com e crie uma conta gratuita
2. Instale o CLI:
   ```
   npm install -g vercel
   ```
3. Na pasta com os arquivos, rode:
   ```
   vercel
   ```
4. Siga as instruções → em 2 minutos você terá um link como:
   `https://eldorado-dashboard.vercel.app`

---

## 📱 Instalar no celular (Android)

1. Abra o link no **Chrome**
2. Toque nos **3 pontinhos** no canto superior direito
3. Selecione **"Adicionar à tela inicial"**
4. Confirme → o app aparece com ícone dourado na tela inicial ✅

## 📱 Instalar no celular (iPhone)

1. Abra o link no **Safari** (obrigatório no iPhone)
2. Toque no ícone de **compartilhar** (quadrado com seta)
3. Selecione **"Adicionar à Tela de Início"**
4. Confirme → app instalado ✅

---

## 💻 Instalar no PC (Chrome/Edge)

1. Abra o link no navegador
2. Na barra de endereços, clique no ícone de **instalar** (monitor com seta)
   - Ou vá em Menu → "Instalar Eldorado Dashboard"
3. Confirme → abre como janela separada, sem barra do navegador ✅

---

## 📶 Funciona offline?

Sim! Após a primeira abertura, o app funciona sem internet.
- Os seus dados ficam salvos localmente no dispositivo
- A cotação USD/BRL só atualiza quando há conexão
